Clinical Decision Support (CDS)

What this site does

This is a minimal teaching app for BIOM9450 showing a full stack: a Node.js/Express backend with MySQL, and a tiny HTML/JS frontend. It supports role-aware access to patient data and on-demand LLM report generation (Mistral).

  • POST /register_user – create an account and receive a JWT.
  • POST /login_user – log in to get a JWT.
  • GET/POST /patients – list/create patients (role aware).
  • GET /patients/search – search by id or name/DOB.
  • POST /reports/generate – ask the LLM to write a report for a patient.
